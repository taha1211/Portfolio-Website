Kapability

What is Kapability?

Kapability is a Progressive Web App (PWA) aimed at fighting inequality within the job market by creating a platform for those with disabilities to find employment easier and more reliably. Using employer-employee pairing techniques, the application shows users job-opportunities that are guaranteed to accomodate their disability and special requirements.

Currently, Kapability is undergoing intensive development and investment to meet its target launch date of September 2018. Information specific questions can be redirected and viewed on Kapability's own website.

Development Goals

Universal Implementation

Using a PWA framework allows Kapability to run on a multitude of devices, providing maximum accessibility and ease of development for our software team.

Employee-Employer Channels

Having both employer and employee access for Kapability with different functionality means that two very different system interfaces have to be developed and programmed to make the application functional.

Server-Side Efficiency

In order to satisfy the different calls that Kapability has to make to the server efficiently and quickly, our back-end code needs to be optimized for efficiency and speed, able to function consistently on a multitude of devices.

Application Search

While simple from the outside, search has to encorporate different criteria and categorization for not only keywords in the job title, but keywords in organization, and location as well.
